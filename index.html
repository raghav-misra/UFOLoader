Loader will appear in 2 seconds.

  1. Go to /loader.js and copy the code into a JS file and link it with a <script> tag.
  2. Add the following to your JS code:
    • document.body.addEventListener("load", () => UFO.init());
    • const loader = new UFO(element);
    element is an HTMLElement pointing to the loader tag.
    A <style> tag with UFO's CSS will be added when the page loads.
    Also, the empty HTMLElement will have children and attributes added to it.
